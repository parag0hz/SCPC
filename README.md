# SCPC
2025 SCPC AI 챌린지

### 📖 내용

- BLIP2(bootstrapped language-image pretraining) 기반의 대규모 멀티모달 모델 Salesforce/blip2-opt-2.7b를 활용하여 이미지에서 자연어 설명을 생성하는 시스템 구축
- Transformer 기반 이미지-텍스트 사전학습 모델을 fine-tuning 없이 zero-shot 또는 few-shot 형태로 활용해 빠른 프로토타이핑과 실험을 반복 수행
- 사용자의 일상 사진에 대해 선다형 질문의 정답을 예측하는 AI 모델 개발이 목표

### 🙋‍♂️ 역할

- 멀티모달 추론 모델(BLIP2)의 구조 분석 및 사전학습 weight 로딩 오류 해결을 주도함
- 예선 데이터셋 전처리 파이프라인 구축 및 이미지-질문 형태의 batch loader 구성
- 후보 정답 선택지를 포함한 선택형 QA 포맷에 적합하도록 모델의 출력 후처리 로직 개발

### 🎯 결과 및 성과

- 모델의 사전학습 tokenizer 구조를 외부에서 정확히 주입함으로써, 정상적인 inference 파이프라인 구성
- 이미지와 텍스트 질문 쌍에 대해 적절한 정답 선택지를 예측하는 모델 개발
- BLIP2의 zero-shot 추론 능력을 평가하여, 별도의 fine-tuning 없이도 의미있는 정확도 확보

### 📝 프로젝트 회고

- 기존 이미지 분류나 자연어 처리 모델과 달리, 멀티모달 모델은 입력의 복잡성과 tokenizer/processor 구성 요소 간 호환성 문제가 빈번하게 발생하여 구조를 추적하여 해결
- 실제 기업의 문제 해결 과제를 모델링하고, 제한된 시간 내에 성능을 확보하는 과정에서 실전형 AI 개발 역량 강화
